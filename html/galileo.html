¿Como tomar una reserva de cero en Galileo?

Void

Para poder voidear una reserva, tenemos que chequear el estatus del ticket
El unico estatus que permite voidear es: OPEN

Status de Ticket:

OPEN Ticket abierto para gestión
- Permite voidear
CHKIN Ticket con check-in hecho
- No permite voidear
ARTP Ticket en control de aeropuerto
- No permite voidear
EXCH Ticket cambiado
- No permite voidear

El comando de void es: TRV/(nrodetkt)

Luego de voidear, el sistema nos devolverá las siguientes respuestas:

Vamos a ignorar y volver a desplegar el ticket para chequear el estatus:
El comando para ignorar es: IR
Es importante destacar que el comando SOLO void el ticket, no elimina los segmentos.
Es mala práctica dejar segmentos/tramos inactivos.
Siempre vamos a limpiar el itinerario si no vamos a emitir un ticket.
El comando para limpiar el pnr es: XI

El gds nos informa que el itinerario ha sido limpiado, y el ticket ha sido voideado.
Veremos solo el nombre en el pnr.

Datos y Comandos importantes

Gestión de aéreos

Para poder tener una mayor rentabilitad en cada pedido es importante conocer el proceso correcto de las herramientas de cobro
 Cuando procesamos un cobro por nuestro carrito de admin, estamos pagando un costo por el uso de la plataforma bajo nuestro numero de comercio.
Este importe es porcentual, es decir cuanto mayor sea el importe que procesemos en el carrito, mayor es el costo. O sea, menos ganancia de esa reserva
La gestión entonces se debe realizar de la siguiente manera:
Se cobra por separado:
Monto aéreo ↠ con el comercio de la aerolínea
Cargos de gestión Atrápalo + Seguros/Asientos ↠ con el comercio de Atrápalo

¿Qué significa UATP?

Son los cobros que se realizan con el comercio de la compañía aérea
Dependiendo del tipo de tarjeta, se pueden:
Se pueden cobrar directamente desde el GDS
Se pueden cobrar por Lapos (Payway). Dar click acá para ver el proceso
Se pueden cobrar por Fiserv (Master). Dar click acá para ver el proceso
Al momento de procesar un cobro UATP, es súper importante verificar el número de comercio por donde se va a cobrar

En algunas ocasiones podemos encontrarnos con compañias aereas que no tienen numero de comercio para procesar ciertos tipos de tarjetas.
O bien que solo permitan la emisión de tickets CASH.
Si este escenario se presenta vamos a cobrar por admin (con nuestro numero de comercio) y emitir cash.
